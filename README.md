# Angular Elements 8 Demo

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 8.3.26.

## 測試方法

1. 安裝套件

    ```sh
    npm install
    ```

2. 建置與預覽

    ```sh
    npm run build
    ```

    連到 <http://localhost:5000/> 進行測試！
